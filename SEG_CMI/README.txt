LaTeX template for SEG 2015 Annual Meeting expanded abstracts.

All files in this directory are extraced from the SEGTeX package 
available at

	http://www.ahay.org/wiki/SEGTeX

Example LaTeX files are segabs_example.ltx and segabs_final.ltx.

Process with

	pdflatex segabs_example.ltx
	bibtex segabs_example
	pdflatex segabs_example.ltx
	pdflatex segabs_example.ltx
	pdflatex segabs_final.ltx

The results should be equivalent to 

        http://reproducibility.org/wikilocal/docs/segabs_example.pdf
	http://reproducibility.org/wikilocal/docs/segabs_final.pdf
